O que é box (caixa) no web dev?

Box model, ou modelo de caixa, é uma estrutura na qual todo elemento visual de um site está. É com ele que será realizada a manipulação do design da página, a partir do CSS. Por exemplo:

Olá, mundo!

No texto acima vemos um exemplo da delimitação de uma caixa, porém com o texto centralizado. Nesse exemplo:

  • margin: para determinar a margem esquerda "margin-left" e direita "margin-right" (a fim de dar um espaço a mais das "paredes" do site) e o inferior "margin-bottom" (para dar um espaço entre o "<h1>" e o "<p>");
  • text-align: para centralizar o texto "text-align: center;", já que por padrão ele vem alinhado à esquerda;
  • border: com a finalidade de demonstrar a delimitação da "box" do elemento "h1". Essa borda faz parte internamente da "box" original do elemento;
  • outline: com o objetivo de demonstrar a possibilidade de um traçado e que essa outline é a parte mais interna da margin do elemento;
  • background-color: para demonstrar visualmente toda a àrea que a box desse elemento cobre.
  • width/height: para determinar a altura e a largura da "box"

Caso o elemento estiver dentro de uma outra box, "div", por exemplo, a centralização se daria por margin: auto;.

Tipos de caixa

Box-level

Esse tipo de caixa tem a propriedade de quebra de linha, seja de um parágrafo ou qualquer outro conteúdo do site. A caixa sempre ficará abaixo de algum outro tipo de caixa. Exemplos:

  • <div>
  • <h1>...<h6>
  • <p>
  • <header>, <main>, <footer>
  • <section>, <article>, <aside>
  • <nav>
  • <form>, <video>

Inline-level

Já esse consiste em não haver a quebra de linha. A forma se adequa ao tamanho do conteúdo. Exemplos:

  • <a>
  • <strong>, <em>
  • <sup>, <sub>
  • <button>, <input>
  • <select>
  • <code>
  • <small>

Grouping tags

As grouping tags são uma forma semântica de hierarquia entre tags. Elas ajudam a não utilizar sempre "id" e "class", separando pelo significado dentro do código que aquele elemento terá.

Elas também influenciam no SEO (Search Engine Optimization).

Exemplos:

  • <header>, <main>, <footer>
  • <section>, <article>, <aside>
  • <nav>
  • <span>

Box shadow

Para colocar sombras em "box", temos a declaração "box-shadow: (height)px (width)px (blur)px (spread)px color;".

  • heigth: altura;
  • width: largura;
  • blur: desfoque, suavização;
  • spread: espalhamento.

Border radius

O border radius diz a respeito do arredondamento dos vértices de um "box". A declaração usada no código para tal é border-radius: (top-left)px (top-right)px (bottom-right)px (bottom-left)px;.

Caso queira bordas com mesmo raio, basta fazer somente border-radius: (raio)px;. Ou, caso queira fazer uma intercalação entre as diagonais, uma diagonal com 2 bordas iguais e outra com 2 bordas iguais, basta fazer border-radius: (top-left && bottom right)px (top-right && bottom-left)px;.
